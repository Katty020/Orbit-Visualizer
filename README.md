# Orbit Visualizer

A fullscreen, interactive orbit art playground built with Pygame. Create glowing planets, tune time, and capture cosmic screenshots.

## Run

```bash
python main.py
```

## Controls

- Left click: add planet
- Right click: remove nearest planet
- Space: pause/resume
- T: toggle trails
- O: toggle orbits
- L: toggle links
- A: toggle axes
- C: clear trails
- R: reset all planets
- H: hide/show HUD
- S: screenshot (saved to images/)
- +/-: speed up/down
- Q: quit
